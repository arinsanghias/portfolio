Projects at BR Softech

Multiplayer Poker Game

Developed a robust multiplayer poker game utilizing SignalR Core, enabling real-time communication and seamless gameplay for users. The project involved creating a comprehensive casino library and a dynamic DLL that encapsulates core poker game functions and functionalities. This architecture allows for easy integration of various poker variants and enhances the overall gaming experience by ensuring low latency and high responsiveness during gameplay. The game supports multiple players, providing an engaging and interactive environment where users can compete against each other in real-time.

Great Indian Rummy

Created an immersive Rummy game titled "Great Indian Rummy," built on a Unity backend with an SFS (SmartFoxServer) server architecture. This project integrates payment gateways such as Razorpay and PhonePe, facilitating secure and efficient transactions for players. The game features a user-friendly webview that enhances accessibility, allowing players to enjoy the game on various devices without compromising on performance. The design focuses on delivering a smooth user experience, with intuitive controls and engaging graphics that capture the essence of traditional Rummy while incorporating modern gaming elements. Players can join tables, compete in tournaments, and enjoy a social gaming experience with friends and family.
